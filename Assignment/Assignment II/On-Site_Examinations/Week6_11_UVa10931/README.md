# UVa 10931 - Parity

## 1. 題目摘要
給定一個十進位正整數，要把他轉換成「二進位」的字串，並且計算這個二進位裡面總共有幾個 `1`。
當輸入為 `0` 的時候代表測資結束。

## 2. 解題邏輯
這題其實就是在寫以前國中學過的「短除法」：

* **十進位轉二進位**：
  - 用一個 `while (n > 0)` 迴圈不斷把數字除以 2。
  - 每次迴圈抓出餘數 `n % 2`，這個餘數就是二進位的其中一個 Bit。
  - 把算出來的 Bit 透過 `to_string(bit)` 轉成文字，接到字串 `binary` 的後面。
  - 把 $n$ 除以 2 (`n /= 2`)，繼續算下一位。
* **計算 Parity**：
  - 在除的過程中，只要發現餘數是 1，就順便把計數器 `parity` 加 1。
* **字串反轉 (超重要)**：
  - 因為短除法算出來的餘數，第一個出來的其實是二進位的「最低位（最右邊）」，所以我們接出來的字串會是反的。
  - 必須利用 `reverse(binary.begin(), binary.end())` 把整個字串顛倒過來，才是正確的二進位長相。
* 最後按照題目要求的格式印出來就好。

## 3. 注意事項與筆記
- **迴圈終止條件**：`while(cin >> n && n > 0)` 可以同時處理了讀取輸入以及「遇到 0 就結束」這兩個條件。
