# UVa 490 - Rotating Sentences

## 1. 題目摘要
將輸入的多行句子進行「順時針 90 度」旋轉並輸出。
原先的第一行會變成最右邊的那一列，最後一行會變成最左邊的那一列。若各行句子長度不一，較短的句子在旋轉後需要在對應位置補上空白。

## 2. 解題邏輯
這題的核心概念是「二維陣列的翻轉與遍歷」，分為兩個階段：

* **資料儲存與最大長度追蹤**：
    使用 `vector<string>` 來儲存每一行讀入的句子。在讀取的同時，持續更新並記錄所有句子當中的「最大長度 (`max`)」，因為這會是旋轉後輸出的總列數（行數）。
* **矩陣旋轉輸出 (巢狀迴圈)**：
    - **外層迴圈** 控制「直行 (Column)」：從 `j = 0` 跑到 `max - 1`。
    - **內層迴圈** 控制「橫列 (Row)」：因為最後輸入的句子要排在最左邊，所以必須**從下往上**逆向遍歷（從 `lines.size() - 1` 遞減到 `0`）。
* **長度檢查與補齊**：
    在印出字元前，檢查當前句子 `lines[i]` 的長度是否大於 `j`。若符合，則印出該字元 `lines[i][j]`；若長度不足，則印出空白 `" "` 以維持排版。

## 3. 注意事項與筆記
- **越界保護**：C++ 的字串如果存取超過其長度的索引會產生錯誤，因此 `lines[i].length() > j` 這個 `if-else` 判斷是否能 AC 的地方。
- **讀取方式**：必須使用 `getline(cin, str)` 來完整讀取包含空白字元的整行句子。
