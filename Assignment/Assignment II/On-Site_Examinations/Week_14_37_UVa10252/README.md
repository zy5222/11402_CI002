# UVa 10252 - Common Permutation

### 1. 題目摘要

給定兩個由小寫英文字母組成的字串 $A$ 和 $B$。請找出一個最長的字串 $X$，使得 $X$ 的某一種字母排列是 $A$ 的子序列，同時也是 $B$ 的子序列。
最後請將找出的 $X$ 依照「**字母順序 (Alphabetical Order)**」輸出。

### 2. 解題邏輯

雖然題目提到了「子序列 (Subsequence)」，但因為它允許「任何一種字母排列」，實際上字元在原字串中的「先後順序」根本不重要。我們只需要關心**「有哪些字元共同出現，且出現了幾次」**。

1. **頻率統計**：
   建立兩個大小為 26 的整數陣列（代表 `a` 到 `z`），分別統計字串 $A$ 和字串 $B$ 中每個字母出現的次數。
2. **取交集 (最小值)**：
   對於每一個字母，它在共同字串 $X$ 中能出現的最大次數，剛好就是它在 $A$ 和 $B$ 之中出現次數的「**最小值**」。
   * 例如：`a` 在 $A$ 出現 3 次，在 $B$ 出現 1 次，那它最多只能共同貢獻 1 個 `a`。
3. **依序輸出**：
   因為題目要求輸出結果必須依照字母順序排列，我們剛好可以從 `a` 迴圈走到 `z`，把每個字母按照算出的「最小值」次數印出來即可。

### 3. 筆記

* **輸入讀取的陷阱**：
  這題的測資可能會出現「空字串」或「包含空白字元的字串」。如果使用 `cin >> a >> b`，遇到空白就會被截斷或跳過，導致嚴重的錯位 (Wrong Answer)。因此，**必須使用 `getline(cin, string)` 來讀取每一行**。
* **時間複雜度**：
  只需要分別掃描兩個字串一次來統計頻率，再跑一個固定 26 次的迴圈輸出答案。時間複雜度為 $O(|A| + |B|)$，空間複雜度為極小的 $O(1)$（只需常數大小的陣列），效能極佳。
