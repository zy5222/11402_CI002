# UVa 11150 - Cola

## 1. 題目摘要
規則：
1. 一開始買了 $n$ 瓶可樂。
2. 每 3 個空瓶可以換 1 瓶全新的可樂。
3. 最特別的規則：你可以向朋友「借」空瓶來湊滿 3 個，但前提是你喝完換來的可樂後，必須能把空瓶還給朋友。求最後總共最多能喝到幾瓶可樂？

## 2. 解題邏輯
這題可以用 `while` 迴圈慢慢模擬，但我這邊直接用了一行公式 `n + n / 2` 。
背後的邏輯是這樣的：
* 正常來說，3 個空瓶換 1 瓶新可樂（喝完剩 1 個空瓶）。也就是說，你實際上是消耗了 2 個空瓶，來獲得 1 瓶「可以喝的可樂」。
* 既然題目允許「借 1 個空瓶」，那就代表只要我有 2 個空瓶，我就能去借 1 個湊成 3 個，換 1 瓶新的喝掉，然後把剩下的 1 個空瓶還給朋友。
* 結論：**每 2 個空瓶，就能讓我多喝 1 瓶可樂**。
* 所以，一開始買的 $n$ 瓶會產生 $n$ 個空瓶，這些空瓶可以讓我額外換到 $n / 2$ 瓶可樂。總共能喝到的數量就是 `n + n / 2`。

## 3. 注意事項與筆記
- **效能**：這個解法沒有跑任何迴圈，時間複雜度是 $O(1)$。
