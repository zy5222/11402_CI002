# UVa 10170 - The Hotel with Infinite Rooms

## 1. 題目摘要
簡單來說，有一家無限大的飯店。第一批來的旅行團有 $S$ 人，他們會住 $S$ 天；接著下一批旅行團會有 $S+1$ 人，住 $S+1$ 天；再來是 $S+2$ 人住 $S+2$ 天，以此類推。
題目會給初始人數 $S$ 和一個指定的第 $D$ 天，要我們算出在第 $D$ 天的時候，住在飯店的旅行團是多少人。

## 2. 解題邏輯
最直覺的解法就是「時間倒扣」：
* 只要指定天數 `d` 還沒被扣完（`d != 0`，其實更準確來說是只要還沒扣到變負的），就把 `d` 減去當前團體的人數 `s`（因為這批人佔據了 `s` 天）。
* **判斷落點**：如果扣完之後 `d <= 0`，代表題目要求的第 $D$ 天剛好就落在這批 `s` 人的住宿期間內！這時候的 `s` 就是答案，印出來並 `break` 結束迴圈。
* **換下一批人**：如果扣完之後 `d` 還是正的，代表時間還沒到，那就把人數加一 (`s++`)，換下一批更大的旅行團繼續扣。

## 3. 注意事項與筆記
- **資料型態**：題目的天數 $D$ 最大可以給到 $10^{15}$，所以變數 `s` 跟 `d` **一定**要乖乖開 `long long int`，不然會Overflow。
