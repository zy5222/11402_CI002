# UVa 12503 - Robot Instructions

## 1. 題目摘要
有個機器人站在原點 0。給他一連串的指令：
- `LEFT`：往左走一步 (-1)
- `RIGHT`：往右走一步 (+1)
- `SAME AS i`：跟第 i 步做一模一樣的動作
問執行完所有指令後，機器人最後停在坐標多少的位置？

## 2. 解題邏輯
這題的核心是「把過去做過的動作記錄下來」，因為後面隨時會想要抄前面的動作。

* **動作紀錄**：開一個陣列 `a[N]`，遇到 `LEFT` 就存 `-1`，遇到 `RIGHT` 就存 `1`。
* **處理 SAME AS**：
  * 因為 "SAME AS " 這幾個字剛好佔了字串的 index 0 到 7，所以我直接從 index 8 開始抓後面的數字。
  * 用迴圈把字元轉成真正的整數 `idx` (利用 `s[j] - '0'` 的技巧)。
  * 查表：去陣列抓出第 `idx` 步到底是 -1 還是 1，直接複製給當前這一步 `a[i] = a[idx-1]`。
* **算總和**：最後寫個迴圈把 `a` 陣列裡面所有的 -1 和 1 加起來，就是機器人的最終位置。

## 3. 注意事項與筆記
- **換行問題 (Clear Buffer)**：在 `cin >> N;` 讀完指令數量後，一定要補一行 `getline(cin, s);`
  去吃掉換行符號 (Enter)！如果不寫這行，迴圈裡面的第一個 `getline` 會直接讀到空字串。
- *陣列的 Off-by-one 陷阱**：題目說的「第 $i$ 步」是從 1 開始算的（例如 SAME AS 1），但是 C++ 的陣列是從 0 開始存。所以在查紀錄的時候，索引要減一 (`a[idx-1]`) 才能抓到對的資料。
