# UVa 11615 - Family Tree

### 1. 題目摘要

給定一個 $N$ 代的完美二元族譜樹（一共有 $2^N - 1$ 個節點），節點以標準完全二元樹的方式編號（根節點為 1，左子節點為 $2x$，右子節點為 $2x+1$）。
題目給定兩個節點編號 $A$ 和 $B$，並告訴我們這兩個節點其實是「同一個人」。請計算在去除重複的祖先/後代關係後，這個族譜樹中最多有幾個「不重複（Unique）」的人。

### 2. 解題邏輯

這是一題考驗對二元樹結構與等價關係（Equivalence Class）理解的數學題。如果真的去把樹建出來，遇到極大的測資時必定會 Time Limit Exceeded (TLE) 或 Memory Limit Exceeded (MLE)。

1. **二元樹的階層特性**：
   在標準的完全二元樹中，節點 $X$ 的層級（Level）可以透過不斷除以 2 來計算（將根節點設為第 0 層）。
   如果一棵樹總共有 $N$ 代，那麼以節點 $X$ 為根的子樹，其包含的總結點數為 $2^{N - \text{Level}(X)} - 1$。
   
2. **「同一個人」的等價映射**：
   當節點 $A$ 和節點 $B$ 是同一個人時，代表 $A$ 的子樹（祖先們）和 $B$ 的子樹在邏輯上是完全對應的。
   假設 $B$ 的層級比 $A$ 大（也就是 $B$ 位於樹的更深處，距離葉節點較近，因此其子樹較小）。那麼 $B$ 子樹中的**每一個節點**，都會與 $A$ 子樹中的某個節點發生「重疊（Identified）」。
   
3. **結論（計算公式）**：
   因為映射是從「較小的子樹」完全對應到「較大的子樹」上，且層級嚴格遞減，因此絕對不會產生循環映射。我們所「損失」的獨立節點數量，**剛好就等於較小的那棵子樹的節點總數**！
   所以：
   * 最大不重複人數 = (原總節點數) - (較小子樹的總節點數)
   * 原總節點數 = $2^N - 1$
   * 較小子樹的節點數 = $2^{N - \max(\text{Level}(A), \text{Level}(B))} - 1$
   
### 3. 筆記

* **時間複雜度極低**：
  我們只需要計算節點的 Level，時間複雜度僅為 $O(\log(\max(A, B)))$，再配合位元運算，整體複雜度幾乎是 $O(1)$，瞬間就能跑完所有測資。
* **資料型態與位元運算**：
  節點總數最大為 $2^N - 1$，當 $N$ 較大時必定會超過 32-bit 一般整數的極限，所以變數必須使用 `long long` (64-bit integer)。同時在做位元左移時，記得要寫 `1LL << N`，加上 `LL` 後綴才能防止移位時發生整數溢位錯誤。
* **特殊邊界測資 `A == B`**：
  雖然題目通常會給兩個不同的節點，但如果遇到極端測資給了同一個節點（$A=B$），理論上沒有任何節點被額外合併，總數應維持 $2^N - 1$。我們可以在程式碼開頭加上一個判斷來防禦這個 Edge Case。
