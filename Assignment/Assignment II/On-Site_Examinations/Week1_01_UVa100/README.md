# UVa 100 - The 3n + 1 problem

## 1. 題目摘要
給定兩個整數 i 與 j，計算在此區間（包含端點）內所有數字中，擁有最長「循環長度 (Cycle Length)」的數值，並輸出該最大長度。
- 循環規則：若為奇數則 n = 3n + 1，若為偶數則 n = n / 2，直到 n = 1 為止。

## 2. 解題邏輯
這題的實作主要分為三個部分：

* **循環長度計算 (`alg` 函式)**：
    使用 `while` 迴圈模擬 3n+1 的過程。每進行一次運算，計數器 `length` 就加 1，直到數字變成 1。
* **輸入與範圍處理**：
    題目給定的範圍 [ i, j ] 不保證 i < j，因此在進入迴圈前，必須先判斷並使用 `swap(a, b)` 交換數值，確保迴圈能正確從較小值跑向較大值。
* **尋找最大值**：
    遍歷區間內的每一個整數，呼叫 `alg` 函式取得長度，並透過 `max()` 函式不斷更新目前所見的最大循環長度 (`max_leng`)。

## 3. 注意事項與筆記
- **範圍交換**：這題最容易漏掉 i > j 的情況，若沒處理會導致迴圈無法執行。
- **輸出格式**：輸出時必須先印出「原始」輸入的 a 與 b，再印出最大長度。
