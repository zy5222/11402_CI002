# UVa 12405 - Scarecrow

### 1. 題目摘要

在一塊長條形的農田中，農田由 $N$ 個區塊組成，每個區塊要麼是肥沃的土地 (`.`)，要麼是貧瘠的荒地 (`#`)。
我們需要放置稻草人來保護農田。一個稻草人可以保護它所在的區塊以及其「左鄰右舍」（共 3 個連續區塊）。請問至少需要幾個稻草人，才能覆蓋所有的肥沃土地 (`.`)？

### 2. 我的解題邏輯

這是一題標準的「區間覆蓋」問題。為了達成最少稻草人的目標，我們應該採用**「貪婪策略 (Greedy Strategy)」**：

1. **從左到右掃描**：我們從農田的最左邊開始掃描。
2. **發現目標**：當我們遇到第一個需要保護的肥沃土地 (`.`) 時，這就是我們必須處理的對象。
3. **最佳放置位置**：為了讓這個稻草人發揮最大效用，保護到後面更多的農田，我們**不應該**把稻草人放在這個位置，而是把它放在這個位置的**右邊一個區塊**。
   * 舉例：如果肥沃土地在位置 $i$，我們放置稻草人在 $i+1$。這樣這個稻草人可以覆蓋 $i$, $i+1$, $i+2$ 這三個區塊。
4. **跳過已覆蓋區域**：放置完稻草人後，將掃描指標直接跳到 $i+3$ 的位置，繼續尋找下一個未被保護的肥沃土地。

### 3. 筆記

* **為什麼這是最優解？**：
  在處理左邊界未被覆蓋的區塊時，放置稻草人於其右側能最大限度地向右延伸覆蓋範圍。這是一種典型的區間貪婪選擇性質，可以確保在每一步都做出當下最有利的決定。
* **時間複雜度**：
  我們只需要從左到右遍歷一次陣列，時間複雜度為 $O(N)$。對於 $N \le 100$ 的測資，這幾乎是瞬間完成。
* **空間複雜度**：
  只需要一個字串來儲存農田狀態，空間複雜度為 $O(N)$。v
