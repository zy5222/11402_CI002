# UVa 10035 - Primary Arithmetic

## 1. 題目摘要
給定兩個正整數（最多可達 10 位數），模擬加法的過程，並計算總共發生了幾次「進位 (Carry)」。
當輸入為 `0 0` 時結束程式。

## 2. 解題邏輯
這題的主要目的在於「由右至左（低位到高位）」逐位數相加：

* **逐位運算**：使用 `% 10` 取出兩個數字的最後一位，並加上前一次運算的進位值 `c`。
* **判斷進位**：如果該位數的和 `s >= 10`，代表發生進位，計數器 `t` 加 1。
* **更新狀態**：
    - 更新進位值 `c = s / 10` (若有進位則為 1，否則為 0)。
    - 將原本的數字分別 `/ 10` 以處理下一個位數。
* **終止條件**：迴圈會持續執行直到兩個數字都處理完且**不再有剩餘進位值**為止 (`a > 0 || b > 0 || c > 0`)。

## 3. 注意事項與筆記
- **輸出格式**：
    - 0 次進位：`No carry operation.`
    - 1 次進位：`1 carry operation.` (單數)
    - 2 次以上：`x carry operations.` (複數)
- **邊界情況**：最後一位進位到更高位（例如 999 + 1）的情況，必須透過 `c > 0` 的判斷來確保被計入。
