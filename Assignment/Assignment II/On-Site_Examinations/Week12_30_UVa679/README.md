# UVa 679 - Dropping Balls

### 1. 題目摘要

有一棵深度為 $D$ 的完全二元樹（Perfect Binary Tree），每個節點都有一個開關，初始狀態皆為關閉（`false`）。
當一顆球掉落到某個節點時：
* 如果開關是關閉的：球會往「左」子節點掉落，並且該節點的開關切換為開啟（`true`）。
* 如果開關是開啟的：球會往「右」子節點掉落，並且該節點的開關切換為關閉（`false`）。
給定樹的深度 $D$ 以及這是第 $I$ 顆掉落的球，請問這第 $I$ 顆球最後會落在樹底部的哪個葉節點（輸出節點編號）？

### 2. 解題邏輯

如果我們模擬前 $I-1$ 顆球的掉落，時間複雜度會高達 $O(I \cdot D)$，當 $I$ 極大時絕對會超時。但這題其實隱藏著非常漂亮的**數學規律**，我們可以直接推算出第 $I$ 顆球的路徑，完全不需要理會前面的球。

觀察球在任何一個節點的分配情況：
當有 $I$ 顆球經過某個節點時，由於開關每次都會切換，這 $I$ 顆球一定是一半往左、一半往右。
* **如果 $I$ 是奇數**：這顆球會是第 $\frac{I+1}{2}$ 顆往「左」邊走的球。
* **如果 $I$ 是偶數**：這顆球會是第 $\frac{I}{2}$ 顆往「右」邊走的球。

**演算法步驟：**
1. 將球的起點設為根節點，編號 `k = 1`。
2. 往下走 $D-1$ 層，每次檢查當前的 $I$ 是奇數還是偶數：
   * 若為奇數：往左走，節點編號變為 `k * 2`，並且將 $I$ 更新為 `(I + 1) / 2`。
   * 若為偶數：往右走，節點編號變為 `k * 2 + 1`，並且將 $I$ 更新為 `I / 2`。
3. 走完 $D-1$ 層後的 `k` 就是答案。

### 3. 筆記

* **時間複雜度大突破**：
  利用奇偶數的除法規律，我們直接追蹤第 $I$ 顆球的路徑。對於每一筆測資，最多只需要往下走 $D$ 層（題目規定 $D \le 20$）。時間複雜度從 $O(I \cdot D)$ 直接降維打擊變成完美的 $O(D)$。
* **輸入測資的坑**：
  這題的測資格式通常是先給一個整數 $T$ 代表有幾組測資，接著給出 $T$ 行的 $D$ 和 $I$。但有時候檔案最後會多出一個 `-1` 作為結束符號，在讀取時要稍微留意，以免讀到無效的資料。
