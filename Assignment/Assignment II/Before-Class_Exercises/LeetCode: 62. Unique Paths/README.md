# LeetCode 62 - Unique Paths

## 1. 題目摘要
給定一個尺寸為 $m \times n$ 的二維網格。有一機器人位於左上角（座標 $(0, 0)$），欲移動至右下角（座標 $(m-1, n-1)$）。機器人每次僅能向右或向下移動一步。請計算共有多少條不同的獨特路徑可以抵達目的地。

---

## 2. 解題邏輯
本題為經典之**動態規劃（Dynamic Programming, DP）**問題。由於機器人僅能向右或向下移動，抵達任意座標 $(i, j)$ 的路徑數，必然等於抵達其上方座標 $(i-1, j)$ 與左方座標 $(i, j-1)$ 的路徑數總和。

1. **狀態定義與轉移方程式**：
   若使用二維陣列 $dp[i][j]$ 表示抵達 $(i, j)$ 的路徑數，其狀態轉移方程式為：
   $$dp[i][j] = dp[i-1][j] + dp[i][j-1]$$
   
2. **邊界條件（Base Case）**：
   位於第一列（Row 0）與第一行（Column 0）的所有網格，因只能單向直線前往（全向右或全向下），故其路徑數皆為 $1$。亦即 $dp[0][j] = 1$ 且 $dp[i][0] = 1$。

3. **空間複雜度最佳化（Space Optimization）**：
   觀察狀態轉移方程式可發現，計算當前列的狀態時，僅需依賴「上一列的相同行位置」以及「當前列的前一行位置」。因此，可將二維陣列降維至一維陣列 `dp`，長度設為 $n$。
   更新規則簡化為：`dp[j] = dp[j] + dp[j-1]`。
   其中等號右側的 `dp[j]` 相當於上一列的數值，`dp[j-1]` 相當於當前列左方的數值。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    int uniquePaths(int m, int n) {
        // 宣告長度為 n 的一維 DP 陣列，並將初始值皆設為 1 (處理第一列與第一行的邊界條件)
        vector<int> dp(n, 1);
        
        // 從第二列 (i = 1) 與第二行 (j = 1) 開始走訪
        for (int i = 1; i < m; ++i) {
            for (int j = 1; j < n; ++j) {
                // 新的 dp[j] = 舊的 dp[j] (上方) + 更新後的 dp[j-1] (左方)
                dp[j] += dp[j - 1];
            }
        }
        
        // 陣列最後一個元素即為抵達右下角的總路徑數
        return dp[n - 1];
    }
};
