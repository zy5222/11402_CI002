# LeetCode 950 - Reveal Cards in Increasing Order

## 1. 題目摘要
給定一個整數陣列 `deck`，代表一副卡牌。每次操作需自牌堆頂端抽出一張牌並翻開；若牌堆中仍有剩餘的牌，則將當前牌堆頂端的第一張牌移至牌堆最底部。重複此動作直至所有牌皆被翻開。請重新排列原始牌堆，使得最終翻開的牌呈現嚴格遞增排序，並回傳該排列後之陣列。

---

## 2. 解題邏輯
本題若直接進行數值的反向推導較易混淆，最佳解法為利用**佇列（Queue）模擬索引位置**，先推導出每次翻開的牌在原始陣列中對應的「索引」，再依序將由小到大排序好的數值填入該位置。

1. **陣列排序**：
   因最終翻開的順序必須為遞增，故先將輸入之 `deck` 由小到大進行排序，取得所需之數值順序。

2. **索引佇列模擬**：
   * 宣告一個佇列 `q`，並將陣列索引 `0` 至 `n - 1` 依序存入，代表初始未知的牌堆位置。
   * 宣告一個與原陣列等長之結果陣列 `result`。

3. **依序填入數值**：
   逐一讀取已排序之 `deck` 中的每一張牌 `card`（自最小開始）：
   * 自佇列前端取回一個索引 `idx` 並將其彈出（Pop）。此索引即代表目前翻開的牌之原始位置，將 `card` 賦值予 `result[idx]`。
   * 依據題目規則，若佇列不為空，需將佇列當前的前端元素（即下一張牌的索引）取出，並重新推入（Push）佇列尾端，以模擬「將下一張牌移至牌堆底部」之動作。

4. **回傳結果**：
   當迴圈走訪完畢，`result` 即為符合題目要求之起始牌堆排列。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    vector<int> deckRevealedIncreasing(vector<int>& deck) {
        int n = deck.size();
        
        // 將數值由小到大排序
        sort(deck.begin(), deck.end());
        
        // 初始化索引佇列
        queue<int> q;
        for (int i = 0; i < n; ++i) {
            q.push(i);
        }
        
        vector<int> result(n);
        
        // 模擬抽牌與移至底部之過程
        for (int card : deck) {
            // 翻開當前最頂端的牌 (取得其原始索引並賦值)
            int idx = q.front();
            q.pop();
            result[idx] = card;
            
            // 若還有剩餘未翻開的牌，將下一張牌移至牌堆底
            if (!q.empty()) {
                q.push(q.front());
                q.pop();
            }
        }
        
        return result;
    }
};
