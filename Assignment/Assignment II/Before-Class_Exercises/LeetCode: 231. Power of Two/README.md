# LeetCode 231 - Power of Two

## 1. 題目摘要
給定一個整數 $n$，請判斷它是否為 2 的某個次方（即是否存在非負整數 $k$ 使得 $n = 2^k$）。回傳布林值 `true` 或 `false`。

---

## 2. 解題邏輯
本題採用「位元運算」之技巧來判斷整數是否為 2 的冪次。

1. **排除非正數**：
   2 的任何次方皆為正數（$2^0 = 1, 2^1 = 2, \dots$），因此若 $n \le 0$，可直接回傳 `false`。

2. **位元技巧判斷**：
   若 $n$ 為 2 的次方，其二進位表示法必定為「僅包含單一位元為 1，其餘全為 0」（例如 $8 = 1000_2$、$16 = 10000_2$）。
   套用經典位元操作：
   
   $$n \,\&\, (n - 1)$$
   
   此操作將消除 $n$ 最低位之 1。
   * 若 $n$ 僅有一個位元為 1（即為 2 的次方），消除該位元後，結果將等於 $0$。
   * 若 $n$ 包含超過一個位元為 1（非 2 的次方），消除最低位之 1 後，結果仍不等於 $0$。
   
   因此，藉由判斷 `(n & (n - 1)) == 0` 是否成立，即可直接判定 $n$ 是否為 2 的次方。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    bool isPowerOfTwo(int n) {
        if(n <= 0) return false;
        return (n & (n - 1)) == 0;
    }
};
