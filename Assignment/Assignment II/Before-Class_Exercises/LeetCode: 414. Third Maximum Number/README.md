# LeetCode 414 - Third Maximum Number

## 1. 題目摘要
給定一個整數陣列 `nums`，請找出並回傳陣列中**第三大的相異整數**（Third Distinct Maximum Number）。若不存在第三大元素，則回傳陣列中最大的整數。題目要求演算法之時間複雜度應達到 $O(n)$。

---

## 2. 解題邏輯
為符合 $O(n)$ 時間複雜度且僅需 $O(1)$ 額外空間，最佳策略為配置三個變數，分別記錄第一大、第二大與第三大之數值，並於單次遍歷中動態更新。

1. **變數初始化**：
   因陣列元素範圍可能包含 32 位元有號整數之最小值（即 $-2^{31}$），若以 `int` 作為初始值，將無法區分該數值是「預設空值」還是「實際存在的陣列元素」。故採用 64 位元整數型別（`long`），並將初始值設為 `LONG_MIN` 以避免數值衝突。

2. **走訪與更新**：
   逐一讀取陣列中之元素 `num`：
   * 若 `num` 恰等於第一大、第二大或第三大之目前紀錄值，為符合「相異」之定義，直接忽略該元素。
   * 若 `num` 大於第一大：將第一大之數值退位給第二大，第二大之數值退位給第三大，並以 `num` 更新第一大。
   * 若 `num` 介於第一大與第二大之間：將第二大之數值退位給第三大，並以 `num` 更新第二大。
   * 若 `num` 介於第二大與第三大之間：直接以 `num` 更新第三大。

3. **回傳結果**：
   走訪結束後，若第三大之變數仍維持初始值 `LONG_MIN`，表示陣列中相異元素不足三個，依題意回傳第一大元素；反之，則回傳第三大元素。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    int thirdMax(vector<int>& nums) {
        long m1 = LONG_MIN;
        long m2 = LONG_MIN;
        long m3 = LONG_MIN;
        
        for (int num : nums) {
            // 忽略重複之數值以確保相異性
            if (num == m1 || num == m2 || num == m3) {
                continue;
            }
            
            // 梯次更新極大值
            if (num > m1) {
                m3 = m2;
                m2 = m1;
                m1 = num;
            } else if (num > m2) {
                m3 = m2;
                m2 = num;
            } else if (num > m3) {
                m3 = num;
            }
        }
        
        // 判斷第三大元素是否存在
        return m3 == LONG_MIN ? m1 : m3;
    }
};
