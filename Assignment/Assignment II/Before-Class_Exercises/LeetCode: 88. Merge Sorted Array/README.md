# LeetCode 88 - Merge Sorted Array

## 1. 題目摘要
給定兩個已排序的整數陣列 `nums1` 與 `nums2`，其初始有效元素數量分別為 $m$ 與 $n$。`nums1` 的實際長度為 $m + n$，其前 $m$ 個元素為欲合併的有效數字，後 $n$ 個元素初始皆為 $0$（作為預留空間）。請將 `nums2` 合併至 `nums1` 中，使合併後的 `nums1` 依然保持遞增排序。本題必須在「原地」（In-place）進行修改，不得宣告額外陣列。

---

## 2. 解題邏輯
若由前往後比較並插入元素，會導致 `nums1` 中原有的元素被覆蓋，從而需要額外的空間或大量搬移操作。為達成 $O(1)$ 的空間限制，最佳解法為採用**三指標由後往前遍歷（Backward Traversal）**之策略。

1. **初始指標設定**：
   * `p1`：指向 `nums1` 中最後一個有效元素的位置（即 $m - 1$）。
   * `p2`：指向 `nums2` 中最後一個有效元素的位置（即 $n - 1$）。
   * `p`：指向 `nums1` 陣列的最末端，即欲填入元素之目標位置（即 $m + n - 1$）。

2. **比較與填入**：
   * 比較 `nums1[p1]` 與 `nums2[p2]` 之大小，將較大者放置於 `nums1[p]`。
   * 將被選中的指標（`p1` 或 `p2`）以及目標指標 `p` 遞減。
   * 持續此過程，直到 `p1` 或 `p2` 其中一個小於 $0$（即其中一個陣列之有效元素已全數安放完畢）。

3. **處理剩餘元素**：
   * 若 `p1` 先小於 $0$，代表 `nums2` 中尚有較小的元素未被放入，直接將 `nums2` 剩餘的元素依序填入 `nums1` 前端即可。
   * 若 `p2` 先小於 $0$，則代表 `nums2` 已全數合併完畢，而 `nums1` 原有剩餘的元素本就已排序且在正確位置上，無須進行任何額外操作。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        int p1 = m - 1;
        int p2 = n - 1;
        int p = m + n - 1;
        
        // 當兩個陣列都還有元素時，從後往前比較並填入
        while (p1 >= 0 && p2 >= 0) {
            if (nums1[p1] > nums2[p2]) {
                nums1[p] = nums1[p1];
                p1--;
            } else {
                nums1[p] = nums2[p2];
                p2--;
            }
            p--;
        }
        
        // 若 nums2 還有剩餘元素，將其填入 nums1 的前端
        // (若 nums1 有剩餘則不需處理，因為已經在正確位置上)
        while (p2 >= 0) {
            nums1[p] = nums2[p2];
            p2--;
            p--;
        }
    }
};
