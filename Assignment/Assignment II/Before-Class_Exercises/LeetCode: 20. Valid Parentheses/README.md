# LeetCode 20 - Valid Parentheses

## 1. 題目摘要
給定一個僅包含字元 `'('`、`')'`、`'{'`、`'}'`、`'['` 與 `']'` 的字串 `s`，請判斷該輸入字串是否為合法括號序列。
合法字串必須滿足以下條件：
1. 左括號必須由相同類型的右括號閉合。
2. 左括號必須以正確的順序閉合。
3. 每個右括號皆有一個對應之同類型左括號。

---

## 2. 解題邏輯
本題為資料結構「堆疊（Stack）」之最經典應用。括號的匹配具備「後進先出（LIFO, Last-In-First-Out）」之特性，亦即最後出現的左括號，必須最先被其對應的右括號閉合。

1. **宣告堆疊**：
   建立一個字元型態之堆疊 `st`，用以暫存尚未被匹配的左括號。

2. **字串走訪與匹配**：
   由左至右逐一讀取字串 `s` 中的每一個字元 `c`：
   * 若 `c` 為左括號（`'('`、`'{'` 或 `'['`）：將其推入堆疊中。
   * 若 `c` 為右括號（`')'`、`'}'` 或 `']'`）：
     * 首先檢驗堆疊是否為空。若為空，代表此右括號前方無任何可供匹配之左括號，字串不合法，直接回傳 `false`。
     * 若堆疊不為空，則取出堆疊頂部之字元（即最近一次存入的左括號），並將其自堆疊中彈出（Pop）。
     * 檢驗取出的左括號與當前的右括號是否屬於相同類型。若類型不吻合，字串不合法，回傳 `false`。

3. **終止條件與最終檢查**：
   當整個字串走訪完畢後，必須檢查堆疊是否為空。若堆疊為空，代表所有左括號皆已順利尋獲對應之右括號，回傳 `true`；若堆疊中仍有殘留之左括號，則代表缺少相應之右括號予以閉合，回傳 `false`。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;
        
        for (char c : s) {
            // 若為左括號，推入堆疊
            if (c == '(' || c == '{' || c == '[') {
                st.push(c);
            } 
            // 若為右括號，進行匹配檢驗
            else {
                if (st.empty()) {
                    return false;
                }
                
                char top = st.top();
                st.pop();
                
                if ((c == ')' && top != '(') ||
                    (c == '}' && top != '{') ||
                    (c == ']' && top != '[')) {
                    return false;
                }
            }
        }
        
        // 最終確認堆疊是否清空
        return st.empty();
    }
};
