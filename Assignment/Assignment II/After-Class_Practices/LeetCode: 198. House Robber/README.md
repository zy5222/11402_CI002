# LeetCode 198 - House Robber

## 1. 題目摘要

有一排房屋，每間房屋內都有一定金額的現金，存放於陣列 `nums` 中。

由於相鄰的房屋裝有防盜系統，因此**不能偷取相鄰的兩間房屋**。

請計算在不觸發警報的情況下，最多可以偷到多少金額。

例如：

```text
Input:
nums = [1,2,3,1]

Output:
4
```

Explanation：

```text
偷第 1 間與第 3 間：

1 + 3 = 4
```

```text
Input:
nums = [2,7,9,3,1]

Output:
12
```

Explanation：

```text
偷第 1、3、5 間：

2 + 9 + 1 = 12
```

---

## 2. 解題邏輯

本題使用 **Dynamic Programming（動態規劃）**。

建立一維陣列：

```cpp
dp[i]
```

表示：

> 偷到第 `i` 間房屋時，前 `i` 間房屋所能獲得的最大金額。

---

### Step 1：初始化

若只有一間房屋：

```cpp
dp[0] = nums[0];
```

若有兩間房屋：

```cpp
dp[1] = max(nums[0], nums[1]);
```

因為不能同時偷相鄰兩間，只能選金額較大的那一間。

---

### Step 2：狀態轉移

對於第 `i` 間房屋，有兩種選擇：

1. **不偷第 `i` 間**  
   最大金額為：

```cpp
dp[i - 1]
```

2. **偷第 `i` 間**  
   則前一間不能偷，因此最大金額為：

```cpp
dp[i - 2] + nums[i]
```

取兩者最大值：

```cpp
dp[i] = max(dp[i - 1], dp[i - 2] + nums[i]);
```

---

### Step 3：回傳答案

最後：

```cpp
dp[n - 1]
```

即為可以偷得的最大金額。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    int rob(vector<int>& nums) {
        int n = nums.size();

        if (n == 1)
            return nums[0];

        vector<int> dp(n);

        dp[0] = nums[0];
        dp[1] = max(nums[0], nums[1]);

        for (int i = 2; i < n; i++) {
            dp[i] = max(dp[i - 1], dp[i - 2] + nums[i]);
        }

        return dp[n - 1];
    }
};
```

---

## 4. 複雜度分析

| 項目 | 複雜度 |
|------|--------|
| 時間複雜度 | **O(n)** |
| 空間複雜度 | **O(n)** |

其中：

- `n` 為房屋數量。

每間房屋只會計算一次，因此時間複雜度為 **O(n)**。

DP 陣列需儲存 `n` 個狀態，因此空間複雜度為 **O(n)**。

> **補充：** 由於每次只會用到前兩個狀態，因此可利用滾動變數將空間複雜度最佳化為 **O(1)**。

---

## 5. 重點整理

- 使用 **Dynamic Programming（DP）** 求解最大收益。
- `dp[i]` 表示前 `i` 間房屋可偷得的最大金額。
- 狀態轉移公式為：
  ```cpp
  dp[i] = max(dp[i - 1], dp[i - 2] + nums[i]);
  ```
- 每間房屋都有「偷」或「不偷」兩種選擇，取收益較大的方案。
- 此方法時間複雜度為 **O(n)**，是本題最經典且效率最佳的解法。
