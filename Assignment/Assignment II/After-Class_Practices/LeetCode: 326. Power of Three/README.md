# LeetCode 326 - Power of Three

## 1. 題目摘要

給定一個整數 `n`，判斷它是否為 **3 的冪次（Power of Three）**。

若存在某個整數 `x`，使得：

```text
n = 3^x
```

其中 `x >= 0`，則回傳 `true`；否則回傳 `false`。

例如：

```text
Input: 27
Output: true

Input: 45
Output: false

Input: 1
Output: true
```

---

## 2. 解題邏輯

本題最直觀的方法是 **不斷除以 3**。

### Step 1：排除非法情況

若

```cpp
n <= 0
```

則一定不是 3 的冪次，直接回傳 `false`。

---

### Step 2：持續除以 3

只要目前數字可以整除 3：

```cpp
while (n % 3 == 0)
```

就持續：

```cpp
n /= 3;
```

若 `n` 原本就是

```
27

27 → 9 → 3 → 1
```

最後一定會變成 `1`。

---

### Step 3：判斷是否為 1

若最後

```cpp
n == 1
```

表示原本為

```
3^0
3^1
3^2
3^3
...
```

其中之一，因此回傳 `true`。

否則回傳 `false`。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    bool isPowerOfThree(int n) {
        if (n <= 0)
            return false;

        while (n % 3 == 0)
            n /= 3;

        return n == 1;
    }
};
```

---

## 4. 複雜度分析

| 項目 | 複雜度 |
|------|--------|
| 時間複雜度 | **O(log₃ n)** |
| 空間複雜度 | **O(1)** |

- 每次迴圈都將 `n` 除以 3，因此最多執行 **log₃ n** 次。
- 僅使用固定數個變數，因此空間複雜度為 **O(1)**。

---

## 5. 重點整理

- 若 `n <= 0`，一定不是 3 的冪次。
- 不斷將 `n` 除以 3，直到無法整除。
- 若最後結果為 `1`，表示 `n` 可以表示成 `3^x`。
- 此方法簡單直觀，且只需 **O(log₃ n)** 時間與 **O(1)** 額外空間。
