# LeetCode 670 - Maximum Swap

## 1. 題目摘要

給定一個非負整數 `num`，你可以**最多交換一次**其中任意兩個數字的位置。

請回傳交換後所能得到的**最大整數**；若不需要交換即可得到最大值，則直接回傳原數字。

例如：

```text
Input: 2736
Output: 7236

Input: 9973
Output: 9973
```

---

## 2. 解題邏輯

本題的關鍵是：**希望越高位的數字越大，因此應優先交換前面的較小數字。**

### Step 1：將整數轉成字串

為了方便交換數字，先將整數轉成字串：

```cpp
string s = to_string(num);
```

之後即可直接交換字元。

---

### Step 2：記錄每個數字最後出現的位置

建立大小為 10 的陣列：

```cpp
vector<int> last(10);
```

遍歷整個字串：

```cpp
last[s[i] - '0'] = i;
```

例如：

```
2736

數字：2 7 3 6
位置：0 1 2 3
```

得到

```
last[2] = 0
last[3] = 2
last[6] = 3
last[7] = 1
```

---

### Step 3：由左往右尋找可交換的位置

依序檢查每個數字。

假設目前是

```
2
```

就嘗試找

```
9、8、7、6、5、4、3
```

是否有更大的數字出現在它的右邊。

若存在：

```cpp
last[d] > i
```

表示可以交換。

---

### Step 4：交換一次後立即回傳

題目限制**最多交換一次**。

因此找到第一組最佳交換後：

```cpp
swap(s[i], s[last[d]]);
```

即可回傳：

```cpp
return stoi(s);
```

若整個過程都找不到更大的數字可以交換，表示原本就是最大值，直接回傳 `num`。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    int maximumSwap(int num) {
        string s = to_string(num);
        vector<int> last(10);

        // 記錄每個數字最後出現的位置
        for (int i = 0; i < s.size(); i++) {
            last[s[i] - '0'] = i;
        }

        // 尋找最佳交換位置
        for (int i = 0; i < s.size(); i++) {
            for (int d = 9; d > s[i] - '0'; d--) {
                if (last[d] > i) {
                    swap(s[i], s[last[d]]);
                    return stoi(s);
                }
            }
        }

        return num;
    }
};
```

---

## 4. 複雜度分析

| 項目 | 複雜度 |
|------|--------|
| 時間複雜度 | **O(n)** |
| 空間複雜度 | **O(1)** |

- 建立 `last` 陣列需遍歷一次字串。
- 每個位置最多檢查 10 個數字（0～9），屬於常數次操作，因此整體時間複雜度仍為 **O(n)**。
- `last` 陣列大小固定為 10，因此空間複雜度為 **O(1)**。

---

## 5. 重點整理

- 將整數轉成字串，方便交換數字。
- 使用 `last` 陣列記錄每個數字最後一次出現的位置。
- 從左到右尋找第一個可以與右側更大數字交換的位置。
- 優先選擇最大的數字（9 → 0）且最後出現的位置，可得到最大結果。
- 找到最佳交換後立即回傳，若沒有可交換的位置則回傳原數字。
