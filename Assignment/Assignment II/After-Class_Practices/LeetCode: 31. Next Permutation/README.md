# LeetCode 31 - Next Permutation

## 1. 題目摘要

給定一個整數陣列 `nums`，請將其重新排列成**下一個字典序（Lexicographically Next）排列**。

若存在下一個更大的排列，則修改陣列為該排列；若不存在（目前已是最大排列），則將陣列重新排列成最小排列（升序）。

要求：

- **必須原地（In-place）修改陣列。**
- **只能使用 O(1) 額外空間。**

例如：

```text
Input: nums = [1,2,3]
Output: [1,3,2]
```

```text
Input: nums = [3,2,1]
Output: [1,2,3]
```

```text
Input: nums = [1,1,5]
Output: [1,5,1]
```

---

## 2. 解題邏輯

本題利用 **Greedy（貪心）** 與 **反轉（Reverse）** 的技巧完成。

### Step 1：從右往左找到第一個遞減的位置

由右往左尋找第一個滿足：

```cpp
nums[i] < nums[i + 1]
```

的位置。

例如：

```
1 2 7 4 3 1
    ↑
```

找到 `2`。

若找不到，代表整個陣列為遞減排列，例如：

```
5 4 3 2 1
```

表示目前已是最大排列，直接反轉整個陣列即可。

---

### Step 2：從右往左找到第一個比 nums[i] 大的數

再次從右往左搜尋：

```cpp
nums[j] > nums[i]
```

找到最右側符合條件的數字。

例如：

```
1 2 7 4 3 1
      ↑
```

找到 `3`。

---

### Step 3：交換兩個數字

交換：

```cpp
swap(nums[i], nums[j]);
```

例如：

```
1 3 7 4 2 1
```

---

### Step 4：反轉後半段

交換完成後，右半部仍為遞減排列：

```
7 4 2 1
```

將它反轉：

```cpp
reverse(nums.begin() + i + 1, nums.end());
```

得到：

```
1 3 1 2 4 7
```

這就是下一個字典序排列。

---

## 3. 程式碼實作

```cpp
class Solution {
public:
    void nextPermutation(vector<int>& nums) {
        int n = nums.size();
        int i = n - 2;

        // 找到第一個遞減的位置
        while (i >= 0 && nums[i] >= nums[i + 1])
            i--;

        if (i >= 0) {
            int j = n - 1;

            // 找到右側第一個比 nums[i] 大的數
            while (nums[j] <= nums[i])
                j--;

            swap(nums[i], nums[j]);
        }

        // 將後半段反轉
        reverse(nums.begin() + i + 1, nums.end());
    }
};
```

---

## 4. 複雜度分析

| 項目 | 複雜度 |
|------|--------|
| 時間複雜度 | **O(n)** |
| 空間複雜度 | **O(1)** |

- 最多遍歷陣列兩次，因此時間複雜度為 **O(n)**。
- 僅使用固定數個變數，符合題目要求的 **O(1)** 額外空間。

---

## 5. 重點整理

- 從右往左找到第一個遞減的位置 `nums[i] < nums[i+1]`。
- 再從右往左找到第一個比 `nums[i]` 大的數字。
- 交換兩者。
- 將 `i` 後方所有元素反轉，使其成為最小排列。
- 若找不到遞減的位置，表示目前已是最大排列，直接反轉整個陣列即可得到最小排列。
