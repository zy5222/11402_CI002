## Repository Description
This repository is dedicated to the **C++ Practicum** course. It serves as a living document where I systematically record my learning progress, technical growth, and reflections throughout the semester.

### Overview
Through structured classroom training, I focus on solving **UVA Online Judge** problems to bridge the gap between theory and practical implementation. By leveraging GitHub for **version control** of my solutions, I track:
* **Logic Design**: How I approach and decompose complex problems.
* **Debugging & Refactoring**: Documenting bugs, fixes, and code improvements.
* **Retrospectives**: Analyzing mistakes to refine my programming mental models.

### AI-Assisted Learning (Gemini)
In this repository, I integrate **Gemini** as a collaborative AI partner to enhance my learning efficiency. My AI workflow includes:
* **Code Review**: Asking Gemini for suggestions on improving code readability and performance.
* **Edge Case Analysis**: Discussing potential hidden test cases that I might have missed.
* **Concept Deep-Dives**: Using AI to clarify complex C++ features or STL functionalities.
* **Prompt Engineering**: Refining how I ask technical questions to get precise, actionable insights.

* **CPE Preparation**: Build a systematic workflow to solve problems efficiently during the **CPE**.
